# ExpandRecipe-Improved

ExpandRecipe-Improved is a MelonLoader mod for the game **Schedule I**. It is an updated and heavily patched version of the original `Schedule1_ExpandRecipe` mod created by *robbmanes*. 

This mod enhances the in-game Product Manager App on your phone by expanding recipe trees, allowing you to see all base ingredients required for a product at a glance, while also injecting vital cost and profit calculations.

## ✨ Features

* **Full Recipe Expansion:** Instantly view the entire crafting tree for any product on your phone, bypassing the need to click through multiple sub-menus.
* **Profit & Cost Breakdown:** Hovering over the final product in the expanded recipe now displays a detailed tooltip showing the total base cost of all ingredients and your calculated net profit.
* **Infinite Loop Protection (Cycle Detection):** Fixes a critical bug from the original mod where cyclical recipes (recipes that loop back into themselves) would cause infinite recursion, crashing the game or UI.
* **UI & Aspect Ratio Fixes:** Automatically cleans up the expanded UI to prevent overlapping icons when clicking between products, and forces image aspect ratios to be preserved so icons are no longer stretched.

## 📥 Installation

1. Ensure you have [MelonLoader](https://melonwiki.xyz/#/) installed for your game.
2. Download the latest `.rar` release file from the Releases tab.
3. Locate your game's main installation folder (`Schedule I`).
4. Drop the `.rar` file directly into folder (`Schedule I`).
5. Right-click the `.rar` file and select **"Extract Here"** (this will automatically place the mod files into your `Mods` folder).
6. Launch the game!
